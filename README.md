# 🏋️ Body Health System

## 📌 Overview
Body Health System is a software application designed to help users monitor and manage their health and fitness activities efficiently.  
This system provides tools for tracking body metrics, maintaining health records, and improving overall wellness.

---

## 🚀 Features
- 👤 User Management (Register / Login)
- 📊 Body Metrics Tracking (Height, Weight, BMI)
- 🧾 Health Record Management
- 📈 Progress Monitoring
- 🔐 Secure Data Handling

---

## 🛠️ Technologies Used
- **Frontend:** JavaFX / Swing (Desktop UI)
- **Backend:** Java
- **Database:** MySQL
- **Version Control:** Git & GitHub
